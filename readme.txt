Beautiful before/after image comparison slider for Gutenberg. Perfect for showcasing transformations, renovations, and results.

== Description ==

**Smart Before After** is a Gutenberg block that helps you create stunning before/after image comparisons for your WordPress website.

Perfect for:

* **Photography** - Show photo editing before and after
* **Home Renovation** - Display renovation transformations
* **Beauty & Fitness** - Showcase treatment results
* **Web Design** - Present website redesigns
* **Product Demos** - Compare product versions

### Features

* **2 Orientations** - Horizontal and vertical slider
* **2 Interaction Modes** - Drag or hover to compare
* **3 Handle Styles** - Arrows, circle, and line designs
* **Fullscreen Mode** - View comparisons in lightbox
* **Smart Labels** - Customizable before/after labels with smart clipping
* **Caption Support** - Add descriptions above or below slider
* **Scroll Animation** - 5 animation types on scroll
* **Smart CTA Trigger** - Show call-to-action after user interaction
* **Fully Responsive** - Looks great on all devices
* **Touch Support** - Works perfectly on mobile devices
* **Keyboard Accessible** - Full keyboard navigation support
* **Lightweight** - No jQuery, pure vanilla JavaScript

For more information, visit the [GitHub repository](https://github.com/cartfixwp/smart-before-after).

== Installation ==

1. Install and activate the plugin from your WordPress dashboard.
2. Edit any page or post with the block editor.
3. Click the (+) button and search for "Smart Before After".
4. Add your before and after images, then customize the slider settings in the sidebar.
